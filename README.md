# HabitSmith

HabitSmith is a modern habit tracking web application built with:

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- Stripe

The project focuses on productivity, streak tracking, analytics, and social habit accountability.

## Features

- Habit tracking
- Streak system
- Analytics dashboard
- Social leaderboard
- Premium subscription support

## Tech Stack

- Next.js
- TailwindCSS
- Zustand
- React Hook Form
- Supabase
- Stripe
